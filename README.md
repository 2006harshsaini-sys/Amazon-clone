# 🛒 Amazon Clone

A responsive Amazon-inspired e-commerce website built using **HTML** and **CSS**. This project replicates the look and feel of Amazon's homepage and showcases modern front-end development skills.

## ✨ Features

* Responsive navigation bar
* Search bar and menu section
* Hero banner section
* Product boxes and categories
* Footer section similar to Amazon
* Clean and user-friendly interface

## 🛠️ Technologies Used

* HTML5
* CSS3
